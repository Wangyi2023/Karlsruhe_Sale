游戏说明

欢迎来到扫雷 :D

请使用桌面端的 Microsoft-Edge, FireFox, Chrome 浏览器打开此页面，移动端页面暂无。
玩家可左键点击屏幕中心区域的方格打开无雷区域，右键点击方格标记为雷，再次左键或者右键点击可取消标记。
控制面板位于侧边栏中，玩家可通过侧边栏的按钮重新开始，修改难度，以及修改背景等。
此游戏使用算法实时在信息面板中会更新当前游戏是否有确定解。点击 Solve 按钮可以自动求解一步， 点击 Solve All 按钮 可以自动求解至游戏结束。
在无解情况下，玩家可点击一个不一定是雷的方格，算法可以在无解并且玩家触雷时重新排布雷的位置，此时所有显现的数字不会被改变。 这能确保玩家不会因为运气差而失败。
最后感谢您的游玩，祝您生活愉快。

Game Guide

Welcome to Minesweeper :D

Please use a desktop browser like Microsoft Edge, Firefox, or Chrome to open this page. A mobile version is not currently available.
Left‑click a cell in the centre grid to open it. Right‑click to mark a cell as a mine. Clicking again will remove the flag.
The control panel is in the sidebar, use the buttons there to restart, change difficulty, switch backgrounds, and access other features.
Use the buttons in the sidebar to restart the game, change difficulty, or switch backgrounds.
The algorithm updates the information panel in real time, showing whether the current board has a certain solution. Click "Solve" to automatically open one step. Click "Solve All" to let the algorithm play until the game ends.
If there is no certain solution, you may open a cell that is not necessarily a mine. If you happen to hit a mine, the algorithm will redistribute the mines without changing any visible numbers. This ensures that players will not lose due to bad luck.
Thank you for playing, and enjoy.
